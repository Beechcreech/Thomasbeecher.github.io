Set up a Git Page

1. Create a github account.
2. Create a new public repository.
3. Enter a name for your repository followed by .github.io ex.name.github.io, then click “Create Repository”.
4. Click on the blue text that says “uploading an existing file”.
5. Somewhere such as vscode create a .html file and add whatever content you’d like.
6. Drag the .html file and any corresponding files into the github project.
7. Commit changes.
